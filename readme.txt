Projeto de site para auxiliar na coleta de lixo

O projeto foi desenvolvido usando html, css e javascript usando o mongoDB para armazenar os dados
Foi dividido em duas partes. a do frontend e a do backend. O backend está funcionando com uma pequena API e comunicação com o mongo.

Requisitos para rodar em localhost:

1 - Ter instalado o mongodb 
2 - ter o nodejs instalado
3 - instalar todas as dependencias contidas no package tanto do backend como do frontend
Como rodar

1 - Certificamos que o mongod está rodando
2 - Executamos o script dev do backend
3 - Executamos o script start do frontend
